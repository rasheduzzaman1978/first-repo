HTML এ CSS এর ব্যবহার

আলহামদুলিল্লাহ, HTML শেখা শুরু করেছি। ৫ জুলাই, ২০২৫ শনিবার বেলা ০২ঃ০০ টার পর থেকে শুরু। খুব ভালো লাগছে। আমি নিয়মিত প্রতিদিন ৪-৫ ঘন্টা সময় দেব বলে নিয়ত করেছি, ইনশাআল্লাহ। বাকি আল্লাহর ইচ্ছা।

আজ ৬ জুলাই, ২০২৫। HTML এ CSS এর ব্যবহার শিখছি। CSS তিনভাবে ব্যবহার করা যায়। যেমনঃ Inline CSS, External CSS এবং Internal CSS আমরা সবসময় External CSS ব্যবহার করার চেষ্টা করব। তা না হলে Internal CSS এবং সবশেষে Inline CSS ব্যবহার করব। Internal CSS লিখতে হয় ট্যাগের মধ্যে।

আমার শখ ভালোভাবে HTML, CSS, JavaScript and Python শিখব এবং এর ব্যবহার কাজে লাগিয়ে অর্থনৈতিকভাবে স্বাবলম্বী হব, ইনশাআল্লাহ।
